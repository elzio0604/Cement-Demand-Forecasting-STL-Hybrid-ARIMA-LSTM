# Cement Demand Forecasting using STL Hybrid ARIMA-LSTM

This study proposes STL Decomposition hybrid ARIMA-LSTM model to improve demand forecasting accuracy in the cement industry.

## Methods
- STL Decomposition
- ARIMA
- LSTM

## Data
The dataset is not publicly available due to confidentiality.
